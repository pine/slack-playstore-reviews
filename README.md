slack-playstore-reviews
-----------------------

:star: Post Google Play reviews to Slack

![](ss.png)

## Requirements

- Node v7.6.0 ~
- Redis

## Getting Started

```
$ git clone https://github.com/pine/slack-playstore-reviews.git
$ cd slack-playstore-reviews
$ npm install
$ SLACK_API_TOKEN=XXX bin/slack-playstore-reviews
```
